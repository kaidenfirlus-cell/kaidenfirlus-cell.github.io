The axis of symmetry of a parabola is a vertical line (x = h) that splits it into two mirror-image halves. For y = ax² + bx + c, the axis is -b ÷ (2a).

The degree is the highest power of the variable with a nonzero coefficient. Example: degree(3x⁴ + 2x²) = 4.

The domain represents all possible x-values that the relation can accept.

The range is the set of all y-values a relation can output. Often described using interval notation.

Skill Development — Chapter 1

Show your own completed, accurate examples for each concept.

1) Transformations of y = a(x − h)² + k

Example: y = -2(x - 3)² + 5

  • a = -2 → reflection over x-axis and vertical stretch
  • h = 3 → right 3
  • k = 5 → up 5

2) When is a relation a function?

A relation is a function when each x-value has exactly one y-value (vertical line test).

3) Domain and Range

Example: y = x² + 2 → Domain: all real numbers; Range: y ≥ 2.

Quadratic Application: Football Height

The model: h(t) = -4.9(t - 1.25)² + 9

A) Initial Height

h(0) ≈ 1.34 m

B) Maximum Height

The vertex → maximum height = 9 m at t = 1.25 s

C) Height at 2.5 seconds

h(2.5) ≈ 1.34 m again (symmetry)

D) Is the ball in the air after 6 seconds?

No — the height becomes negative, meaning the ball already hit the ground.

Domain & Range

0 ≤ t ≤ ~2.5, 0 ≤ h(t) ≤ 9
